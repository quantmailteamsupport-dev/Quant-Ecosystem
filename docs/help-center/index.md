## System Requirements

### Web App

- Chrome 90+, Firefox 90+, Safari 15+, Edge 90+
- JavaScript enabled
- WebAssembly support

### Desktop App

- macOS 12+ (Apple Silicon and Intel)
- Windows 10+ (64-bit)
- Linux: Ubuntu 20.04+, Fedora 36+, or equivalent

### Mobile App

- iOS 16+
- Android 12+

### Storage

- Minimum 500MB free space for app data
- Additional space for synced files
