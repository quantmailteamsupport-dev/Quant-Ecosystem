# Account Management

## Creating Your Account

1. Visit [quant.app/signup](https://quant.app/signup)
2. Enter your email and choose a strong passphrase
3. Encryption keys are generated on your device
4. Verify your email address
5. Save your recovery key securely

## Profile Settings

### Update Profile Information

Go to Settings > Account > Profile:

- Display name
- Profile photo
- Time zone
- Language preference

### Change Email Address

1. Settings > Account > Email
2. Enter new email address
3. Verify via confirmation link
4. Old email receives notification of change

### Change Passphrase

1. Settings > Security > Change Passphrase
2. Enter current passphrase
3. Enter new passphrase (minimum 12 characters recommended)
4. All encryption keys are re-wrapped with new passphrase
5. Other devices will need to re-authenticate

## Security Settings

### Two-Factor Authentication

Enable 2FA at Settings > Security > Two-Factor:

- TOTP (authenticator app)
- Hardware security key (FIDO2)
- Backup codes (save securely)

### Active Sessions

View and manage sessions at Settings > Security > Sessions:

- See all active devices/browsers
- Revoke individual sessions
- "Sign out all devices" for emergencies

### Login History

View recent login attempts at Settings > Security > Login History:

- Successful logins with device/location
- Failed login attempts
- Suspicious activity alerts

## Subscription Management

### View Current Plan

Settings > Billing > Current Plan shows:

- Plan name and features
- Storage usage
- Billing cycle and next payment date

### Upgrade Plan

1. Settings > Billing > Upgrade
2. Choose Pro ($12/mo) or Enterprise
3. Enter payment information
4. Upgrade takes effect immediately
5. Prorated billing for current period

### Downgrade Plan

1. Settings > Billing > Change Plan
2. Select lower tier
3. Review what you will lose
4. Downgrade takes effect at end of billing period
5. Data exceeding new limits is preserved but sync pauses

### Cancel Subscription

1. Settings > Billing > Cancel
2. Access continues until end of billing period
3. Data is preserved but read-only after expiration
4. Re-subscribe anytime to restore full access

## Data Management

### Export All Data

Settings > Data > Export Everything:

- Mail: MBOX format
- Files: Original format in ZIP
- Documents: Markdown, PDF, or DOCX
- Calendar: ICS format
- Contacts: VCF format
- Tasks: JSON format

Export includes all metadata and history. Processing time depends on data volume.

### Delete Specific Data

- Mail: Select messages > Delete
- Files: Right-click > Move to Trash > Empty Trash
- Documents: File > Move to Trash
- Calendar: Click event > Delete

### Storage Cleanup

Settings > Storage > Cleanup:

- View storage breakdown by app
- Find large files
- Clean version history
- Empty all trashes

## Account Deletion

### Permanent Deletion

Settings > Account > Delete Account:

1. Confirm by entering your passphrase
2. All server-side data is permanently deleted
3. Cannot be undone
4. Local device data remains until app is uninstalled
5. 30-day grace period (cancel deletion during this time)

### What Gets Deleted

- All emails and attachments
- All files in Drive
- All documents, sheets, slides
- Calendar events and contacts
- Chat messages and channels
- Task lists and projects
- Photos and albums
- Account credentials and keys

### What is Preserved

- Other users' copies of shared documents (they retain access)
- Messages sent to other users (in their storage)
- Your contributions to shared projects (attributed to "Deleted User")

## Team Administration

### Create a Team

1. Settings > Team > Create Team
2. Set team name and domain
3. Invite members via email
4. Configure team settings and policies

### Manage Members

Admin > Members:

- Invite new members
- Set roles (Admin, Member, Guest)
- Suspend or remove members
- Transfer ownership

### Team Policies

Admin > Policies:

- Password requirements
- 2FA enforcement
- External sharing rules
- Data retention policies
- Allowed integrations
