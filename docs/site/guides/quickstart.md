# Quickstart Guide

Get productive with Quant in 5 minutes.

## 1. Sign Up

Visit [quant.app/signup](https://quant.app/signup) and create your account. Your encryption keys are generated automatically on your device.

## 2. Choose Your Apps

Enable the apps you need:

- **Email**: Quant Mail with E2EE
- **Storage**: Quant Drive for files
- **Writing**: Quant Docs for documents
- **Planning**: Quant Calendar + Tasks

## 3. Import Existing Data

Go to Settings > Import to migrate from:

- Google Workspace
- Microsoft 365
- Apple iCloud

## 4. Install Desktop App

Download the desktop app for the best experience:

```bash
# macOS (Homebrew)
brew install --cask quant

# Linux (Snap)
snap install quant

# Windows (winget)
winget install Quant.Desktop
```

## 5. Set Up Sync

Configure cloud sync (optional, always encrypted):

1. Open Settings > Sync
2. Choose which data to sync
3. Select data region (US, EU, or self-hosted)

## 6. Invite Your Team

For team usage:

1. Go to Settings > Team
2. Invite members via email
3. Set up shared drives and channels
