# Getting Started with Quant

This guide walks you through setting up your Quant account and configuring your first apps.

## Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Or: Quant desktop app for macOS, Windows, or Linux
- Or: Quant mobile app for iOS or Android

## Step 1: Create Your Account

1. Visit [quant.app/signup](https://quant.app/signup)
2. Enter your email address and choose a strong passphrase
3. Your encryption keys are generated locally on your device
4. Verify your email address

> **Important**: Your passphrase is used to derive your encryption keys. Quant cannot reset it. Store your recovery code in a safe place.

## Step 2: Set Up Your Recovery Key

After account creation, you will be prompted to save a recovery key:

1. Download your recovery key file
2. Store it in a secure location (hardware key, safe, etc.)
3. Confirm you have saved it by entering a verification code

## Step 3: Choose Your Apps

Select which apps you want to enable:

- **Communication**: Mail, Chat, Meet
- **Productivity**: Docs, Sheets, Slides, Tasks, Notes, Forms
- **Storage**: Drive, Photos
- **Development**: Code
- **Scheduling**: Calendar

All apps are available on the Free plan. Enable what you need.

## Step 4: Import Your Data

Quant supports importing from:

- Google Workspace (Mail, Drive, Docs, Calendar, Photos)
- Microsoft 365 (Outlook, OneDrive, Word, Excel)
- Apple iCloud (Mail, Drive, Photos, Calendar)
- Standard formats (MBOX, CalDAV, CardDAV, CSV)

Use the import wizard at Settings > Import Data.

## Step 5: Install Desktop & Mobile Apps

Download native apps for the best experience:

- **macOS**: [Download from quant.app/download](https://quant.app/download)
- **Windows**: [Download from quant.app/download](https://quant.app/download)
- **Linux**: Available via Snap, Flatpak, and .deb/.rpm
- **iOS**: [App Store](https://apps.apple.com/app/quant)
- **Android**: [Google Play](https://play.google.com/store/apps/details?id=app.quant)

## Step 6: Configure Sync

By default, Quant uses local-first storage:

1. Go to Settings > Sync & Storage
2. Choose which data to sync to cloud (encrypted)
3. Set up device-to-device sync for offline access
4. Configure sync frequency and bandwidth limits
