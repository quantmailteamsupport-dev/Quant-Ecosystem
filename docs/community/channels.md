# Community Channels

Connect with the Quant community through these channels.

## Discord

**[discord.gg/quant](https://discord.gg/quant)**

Our primary community hub for real-time discussions:

- `#general` - General discussion about Quant
- `#support` - Community help and troubleshooting
- `#feature-requests` - Suggest and vote on features
- `#development` - API and integration discussions
- `#self-hosting` - Self-hosted deployment help
- `#security` - Security discussions and disclosures
- `#showcase` - Share what you have built with Quant

## GitHub Discussions

**[github.com/quant-app/community/discussions](https://github.com/quant-app/community/discussions)**

For longer-form discussions, proposals, and Q&A:

- Announcements - Official updates and releases
- Ideas - Feature proposals and feedback
- Q&A - Technical questions and answers
- Show & Tell - Community projects and integrations
- Polls - Community votes on priorities
